#### 1\) Download

Download the .exe installation file from releases or shortcuts on the top of README file

#### 2\) Run the Installer

Open it as an administrator. <br>
While installing choose NVIDIA acceleration if you have NVIDIA GPU everything else (IGPU, AMD GPUs, INTEL GPUs) CPU only mode (for now!)<br>
Mid-install the terminal will popup - dont worry, it will do all the work and close itself<br>

### 3\) Launch in DaVinci Resolve

1. Open DaVinci Resolve.  
2. Open your Project and Timeline.  
3. Go to Workspace → Scripts → BadWords  
4. Enjoy easier editing\!
